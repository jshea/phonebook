Angular Phonebook

This is a learning and training example application. It is a phonebook/contact manager that implements the basic CRUD elements of business/data applications.

This uses a simple data model and process flow. This gets the model and use case out of the way so we can concentrate on basic CRUD features in a new technology, language or environment.

This project is an example of the following technologies:

  • HTML5, CSS3 and JS5
  • AngularJS - Client application framework
  • Bootstrap - Client application user interface goodies
  • NodeJS - RESTful Web Service
  • MongoDB - Server data storage
  • Responsive Design via Bootstrap
  • Date/Time manipulation via momentjs (date formatting, age calculation)
  • Mapping with Google Maps
  • Angular goodies
    • Grid - ngGrid (sorting, search)
    • Bootstrap directives - ui.bootstrap
      • Tabs
    • Custom directives - Phone formatting
    • Sub controllers
    • Form/Field validation with error formatting & messages (required fields, email formatting)
  • Future
    • Local property storage with LocalStorage
    • Local data storage with IndexedDB
    • Native (hybrid) compilation via Cordova
    • HTML5
      • Application Cache
      • Datalist (state picklist)
The project source and documentation is publicly available for free on github.
